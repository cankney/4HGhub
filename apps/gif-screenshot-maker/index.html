How to Install & Open

01

Download

Click the download button above to retrieve the `GIF_Screenshot_Maker.zip` file containing the application bundle.

02

Extract & Move

Double-click the downloaded ZIP to extract the app. Drag the `GIF Screenshot Maker` application into your `/Applications` directory.

03

First Relaunch

Because the app is built internally, macOS will block it on double-click. **Right-click** on the app in Finder and select **Open**, then click **Open Anyway** in the prompt.

Internal Security Policy

This utility is signed and distributed internally. The system auto-updater connects securely to 4HG Hub to fetch version information. In-app updating will automatically request permissions or prompt for credentials if installed inside the global Applications folder.
